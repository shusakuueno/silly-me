#【アプリ名】
Silly-Me

#【概要】
日常の面白い写真、話を共有するシンプルなWebサービスです。

#【アプリケーション機能一覧】
・記事一覧表示機能

・記事詳細表示機能

・記事CRUD機能

・管理者権限機能

・新規ユーザー登録機能

・ログイン認証機能

・ユーザー編集（氏名、メールアドレス）

・パスワード変更

・画像ファイルアップロード機能

・DBテーブルのリレーション管理

・ページネーション機能

・タイムライン表示機能

・ユーザー退会機能

・ユーザー検索機能

#【使用している技術一覧】
言語：Ruby(2.5.3)、HTML5、CSS3

開発環境：Cloud9（AWS）、Linux、Heroku、git/GitHub

DB：MySQL 14.14

フレームワーク：Rails(5.2.2)、Bootstrap
